# Everything Icon Candidates (Windows Everything Reference)

기존 후보 5개는 제거하고, Windows용 Everything 아이콘(돋보기 심볼) 스타일을 기준으로 다시 5개를 만들었습니다.

1. `doc/icon-candidates/everything-icon-1-classic-orange.svg`
- Everything 기본 오렌지 느낌에 가장 가까운 클래식형.

2. `doc/icon-candidates/everything-icon-2-classic-white.svg`
- 공식 white 변형 느낌을 반영한 모노톤형.

3. `doc/icon-candidates/everything-icon-3-classic-blue.svg`
- 기본 실루엣 유지 + 블루 변형.

4. `doc/icon-candidates/everything-icon-4-flat-orange-gradient.svg`
- 공식 아이콘 팩의 flat/gradient 방향을 반영한 변형.

5. `doc/icon-candidates/everything-icon-5-dark-outline-orange.svg`
- 다크 아웃라인 + 오렌지 본체로 대비를 강화한 변형.

원하면 이 중 하나를 골라 `src-tauri/icons/icon.png`, `src-tauri/icons/32x32.png`, `src-tauri/icons/128x128.png`로 바로 변환/적용해드릴게요.
